Подается число - количество задач, затем вводятся задачи, для каждой пишутся в таком порядке данные: название, приоритет, момент вхождение, время исполнения.
Выводится по очереди названия и время, в которое задача была окончательно исполнена.
В алгоритме используется структура - очередь с приоритетом, реализованная на двоичной куче, которая реализована на векторе. Сложность всех операций на такой очереди логарифмическая.
